# 📊 Sales Analysis Dashboard (Tableau Project)

## 📌 Project Overview
This project focuses on analyzing sales data using Tableau to uncover meaningful business insights. The dashboard helps in understanding sales performance, customer behavior, and product trends, enabling data-driven decision-making.

---

## 🎯 Objectives
- Analyze overall sales performance  
- Identify top-performing products and categories  
- Understand regional sales distribution  
- Track revenue and profit trends  
- Generate actionable insights for business growth  

---

## 🛠 Tools & Technologies Used
- Tableau (Data Visualization & Dashboard Creation)  
- Excel / CSV (Data Source)  

---

## 📊 Dashboard Features
- 📈 Sales Trend Analysis (Monthly & Yearly)  
- 🌍 Region-wise Sales Distribution  
- 🛍 Category & Sub-category Performance  
- 💰 Profit vs Sales Analysis  
- 👥 Customer Segment Insights  

---

## 🔍 Key Insights
- Identified top-performing regions contributing maximum revenue  
- Some categories generate high sales but low profit margins  
- Sales show seasonal trends over time  
- Certain customer segments contribute more to profitability
